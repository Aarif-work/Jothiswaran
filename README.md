# Jothiswaran's Mobile Portfolio

A modern, energetic, mobile-first portfolio website built with React, Tailwind CSS, and Framer Motion.

## Features
- **Mobile-First Design**: Optimized specifically for phone screens.
- **Hero Section**: Full-width HD profile image with gradient overlays.
- **Glassmorphism**: Soft shadows and rounded corners (card-based layout).
- **Smooth Animations**: Scroll-based reveals using Framer Motion.
- **Technical Skills**: Badge-style UI for skills like C++, Java, and HTML.
- **Timeline Education**: Clean representation of academic journey.
- **Sports & Interests**: Highlighting the "Cricket player" and "Sports enthusiast" side.
- **Contact Shortcuts**: Floating Call and WhatsApp buttons for quick interaction.

## Tech Stack
- **Framework**: React (functional components)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Build Tool**: Vite

## Project Structure
```text
/src
  /assets        - Images (profile.jpeg)
  /components    - Hero, About, Skills, Education, Interests, ContactSticky
  App.jsx        - Main layout and routing
  main.jsx       - Entry point
  index.css      - Tailwind imports & global styles
```

## How to Run
1. Ensure you have Node.js installed.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
4. Open `http://localhost:5173` in your browser. (Switch to mobile view in DevTools for the best experience!)
